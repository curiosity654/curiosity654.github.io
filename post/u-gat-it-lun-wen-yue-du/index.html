U-GAT-IT 论文阅读

2020-08-09

Motivation

看完综述后，对 U-GAT-IT 的效果和技术比较感兴趣。本文使用通过几项简洁的改进，综合了无监督、Attention技术，解决了 GAN 中的几个比较经典的问题，增加了网络结构对于不同数据集的泛化能力。另外，论文中二次元头像生成的例子也是吸引我的原因之一。

Contributions

这里简单翻译一下文中列出的贡献

  • 结合 Attention 模块与创新的 AdaLIN 损失函数，设计了无监督图像转换模型
  • Attention 模块可以指导网络对不同领域中进行针对性的变化（不同的地方改动程度不同）
  • AdaLIN 损失函数可以让模型在无需调整结构或超参的情况下适应性地控制形状、材质变化的程度

Model

Generator

将输入图像降采样后输入残差块得到多个特征图，使用辅助分类器ηs\eta_sηs​进行分类判别该图像属于原域或特征域，使用判定得到的权重以及特征图结合得到 attention map，传入 FC 计算 AdaLIN 参数，将特征图结合参数传入残差块构成的解码器，再上采样得到最终结果。

CAM

模型的 Attention 方法受到 CAM 的启发，CAM 是在图像分类网络最后一层使用全局平均池化得到每个 feature map 的均值作为分类 softmax 的输入，训练得到 softmax 的权重后用于加权求和所有特征图，可以得到一个可视化结果。

对于本文的应用，CAM 可以得到判断图像是否为 fake 的 heatmap，作为 Loss 计算的指导

根据论文的描述， AdaIN能够将图像的内容特征转换为风格特征，但是它基于不同特征图间分布独立的假设，相比之下，LN不依赖与分布独立假设，但它对原域中的结构特征保留的不是很好，因此本文提出 AdaLIN，使用参数学习的方法动态调整二者的比例，使其能够很方便地完成多任务的风格迁移。

Loss Function

公式在原论文中都有，这里简述一下目的

  • Adversarial Loss 使得生成的图像分布接近目标域
  • Cycle Loss 减缓模式坍塌问题
  • Identity Loss 使得生成图像的颜色分布接近原图
  • CAM Loss 帮助 GAN 了解需要在哪些地方进行重点改进

Result

是否使用CAM（即 attention）的效果对比，其中图e为加入 CAM 的结果，图f没有 CAM，可以看到 attention map 大多集中于人物面部，未使用 CAM 时，生成图片的轮廓与原图相似性很大，仅仅是全局的材质发生了改变，而使用 CAM 后，生成人物在细节上有了很大改进。
